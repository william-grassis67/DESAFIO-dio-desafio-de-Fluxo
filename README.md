# Desafio Controle de Fluxo

Projeto desenvolvido em Java como parte dos estudos da linguagem para praticar estruturas de controle de fluxo e tratamento de exceções.

## Objetivo

Receber dois números inteiros via terminal, calcular a diferença entre eles e realizar a quantidade correspondente de interações.

Caso o primeiro número seja maior que o segundo, o programa lança uma exceção personalizada (`ParametrosInvalidosException`).

## Tecnologias

* Java
* IntelliJ IDEA

## Conceitos praticados

* Entrada de dados com `Scanner`
* Estruturas condicionais (`if`)
* Laço de repetição (`for`)
* Tratamento de exceções
* Criação de exceções personalizadas
* Organização de classes

## Como executar

1. Clone o repositório.
2. Abra o projeto em uma IDE Java.
3. Execute a classe `Contador`.
4. Informe os dois números solicitados no terminal.
